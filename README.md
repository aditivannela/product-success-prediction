# Product success prediction

## Project Overview

This project analyzes product review data to identify factors associated with product success. The goal is to engineer meaningful features from raw review data to eventually build a machine learning model that predicts whether a product will be successful.

## Technologies Used

- SQL (feature engineering and dataset creation)
- Python
- Pandas
- Matplotlib
- Supabase (PostgreSQL database)

## Data Pipeline

Raw data is stored in two tables:
- products
- reviews

Using SQL, product-level features were generated by aggregating review information.

Pipeline:
Raw Tables → Feature Engineering → Modeling Dataset → Analysis

## Feature Engineering (SQL)

The following features were created for each product:
- review_count – number of reviews per product
- avg_review_rating – average rating
- rating_variability – standard deviation of ratings
- avg_sentiment – average sentiment score from reviews

A binary target variable was created:

is_successful = 1 if
- average rating ≥ 4
- review_count > 100

Otherwise:
- is_successful = 0

## Exploratory Data Analysis

Initial analysis and visualizations were performed in Python using Pandas and Matplotlib to explore relationships between product ratings, review counts, and sentiment scores.
